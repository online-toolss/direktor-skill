## 🚀 Quick Start

```bash
git clone https://github.com/yourusername/direktor-skill
cd direktor-skill
npx create-video@latest my-video
cd my-video
npm install @google/generative-ai @imgly/background-removal zod

# .env file
echo "GEMINI_API_KEY=your_free_key" >> .env
echo "SERPER_API_KEY=your_free_key" >> .env

# Generate!
npx ts-node scripts/generate.ts "60s motivational video about success"
```
